The Catalogue Tool

The catalogue tool is developed as a user-friendly interface to explore mitigation and adaptation solutions at different scales using the NEVERMORE Catalogue developed in WP5. The catalogue of policies, covers different scales following a hierarchical structure for a more efficient exploration of the solutions. Under this hierarchical structure, solutions are organized in primary and secondary sectors.

NEVERMORE technical partners have identified and collected some climate-related policy documents covering different policy sectors (e.g., agricultural, energy, tourism, etc.) at the EU and global levels. Similarly, Case Study leaders were asked to provide CMCC with local policies. From those policy documents, CMCC in collaboration with the technical partners has extracted some relevant adaptation and mitigation measures that were consolidated and then analysed. Thanks to the analysis, the catalogue integrates the following information:

·         The measure description.
·         The scope of the measure (whether it is adaptation or mitigation).
·         The type of measure (if it is land management, for example, or risk management).
·         The actors involved.
·         To what SDGs it contributes.
·         Adaptation and mitigation potentials.
·         Synergies and trade-offs between policies and measures.
·         Drivers and barriers to their applications.

The catalogue that can be used through the tool is developed as an Excel file and as a PostgreSQL relational database for the integration with the others tools of NEVERMORE and to be available for other projects through API protocol allowing that its functionalities can be reused by other application or software.

To explore the catalogue using the tool, a tree is built representing the interlinkages between measures/policies and sectors. The user can navigate through the different sectors by collapsing or expanding the relationship tree to reach the measure about which they want to know more information. On the tool panel, the user will have some basic information about the measures, and then, if the user wants to have more information, there will be a fact sheet available to download with a more detailed explanation of each specific measures. In the fact sheet, you can find all the information per measure as it was collected in the database.

Policies and measures will then be simulated and evaluated in the EU and Local tools.
